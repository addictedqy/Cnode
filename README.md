# Cnode 中文社区
如你所见，一个基于 react 框架 所编写的 cnode 社区

## 工具
1. react
2. react-dom
3. react-redux
4. react-router-dom
5. redux
6. redux-thunk

## React/JSX 规范
<br />[airbnb 的 React/JSX 编码规范](https://github.com/airbnb/javascript/tree/master/react)


## 脚本
启动开发服务器 `npm run dev`
<br /> 如果是 `windows` 系统，则需要重新安装 `node-sass`
<br /> 如果是 `Mac` 系统，则可以直接运行 `npm run dev`
